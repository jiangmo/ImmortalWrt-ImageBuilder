# 固化 dg3399 无线固件配置

根据 dg3399 实机启动日志，板载 WiFi 为 AP6212，对应 Broadcom/Cypress BCM43430 SDIO。

本次修正把 dg3399 构建配置从 `brcmfmac-firmware-43430a0-sdio` 调整为 `cypress-firmware-43430-sdio`，使镜像包含内核实际请求的固件文件：

```text
/lib/firmware/brcm/brcmfmac43430-sdio.bin
```

同时在 dg3399 打包时预置 AP6212 NVRAM 兼容链接：

```text
/lib/firmware/brcm/brcmfmac43430-sdio.txt
/lib/firmware/brcm/brcmfmac43430-sdio.friendlyarm,nanopc-t4.txt
```

这两个链接指向：

```text
brcmfmac43430-sdio.AP6212.txt
```

修正后，新刷 dg3399 固件时无线应能直接加载为 `radio0`，无需再通过 SSH 手动创建固件/NVRAM 链接。
